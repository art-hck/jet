# JET: Angular & Symfony application
## Deploy

####Required to deploy:

| Software  | Download link                 |
|----------:|------------------------------:|
| Docker    | https://www.docker.com/       |
| Node      | https://nodejs.org/en/        |
| Git       | https://git-scm.com/downloads |

####Execute this to deploy the project:
```bash
setup.sh
cd src/frontend/
npm run dev
```
